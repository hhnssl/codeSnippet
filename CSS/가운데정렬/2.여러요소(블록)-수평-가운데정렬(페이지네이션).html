* dot 개수가 달라지고, 크기가 달라져도 전체 가운데 정렬을 유지할 수 있는 방법 (페이지네이션 가능) *
1. display: inline-block
2. 1로 인해 텍스트 취급을 받게되어, 부모인 wrapper에 text-align: center을 주어서 가운데 정렬을 할 수 있다. (margin: 0 auto는 사용 X)
3. dot 들 사이에 있는 공백은 왜 생기는가? 인라인블록으로 인해 dot들이 텍스트화 됨 -> 엔터를 띄어쓰기 한칸으로 인식하기 때문에 생김 -> 해결 방법 dot에 margin: 0 -4px (-4px인 이유는 폰트사이즈에 비례해서 책정되는 값임. 16px일 때 공백은 4px 된다고 생각하면 될듯. -> -0.25em을 주면 자동으로 공백크기가 계산된다.)
